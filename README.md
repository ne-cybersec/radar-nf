# radar-nf

Исследовательский лонгрид **@ne_cybersec**: «Радар.НФ» — единственное в стране и неработоспособное.
Доказательный разбор единственного федерального гражданского приложения для сообщений о БПЛА (ОНФ).

- Публикация: https://ne-cybersec.github.io/radar-nf/
- Один файл `index.html` (CSS + JS без сборки), дизайн-система серии @ne_cybersec.
- OG-карточка: `og-radar.png` (генерируется `bash og.sh`, требует ImageMagick).
